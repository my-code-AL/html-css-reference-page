So what's the difference between padding, border, and margin?

Padding

padding is used to create space around the element in question, but within the confines of the element's border. You can use the "padding" keyword within a css file to specify this value.

Border

The border element can be seen simply as a way to draw a line around an element and its padding.

Margin

This very well may be one of the most useful CSS elements to remember! It allows one to make whitespace between each html element that renders on screen, thereby allowing you to choose how far apart the elements may be from one! another. This is essential to creating a visibly appealing website or webpage!


  • it's worth mentioning that one can define each of the four values that can be used for each of the aforementioned values (i.e. "padding-top","padding-right", etc.), but it is also possible to use two or 4 values simply when using the keyword itself (i.e. padding: 20px 40px; // margin: 20px 30px 40px 50px)
